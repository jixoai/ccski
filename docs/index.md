# ccski

ccski is a CLI + MCP server for discovering, installing, enabling/disabling, and serving Claude/Codex-compatible skills. This site provides two entry points:

- [CLI documentation](/cli/)
- [API documentation](/api/)

## Quick links

- CLI quick start: `npx ccski --help`
- API import: `import { listSkills } from "ccski"`

## Local docs

```bash
pnpm docs:dev
```
